La opcion -L indica al enlazador el directorio donde se encuentran los ficheros de biblioteca

La opcion -I /usr/include indica que se busquen los ficheros de cabecera en el directorio /usr/include 

A simple makefile consists of "rules" with the following shape:
	target: prerequisites
		recipo
A target is usually the name of a filee that is generated by a program
A prerequisite is a file that is used as input to create the target
